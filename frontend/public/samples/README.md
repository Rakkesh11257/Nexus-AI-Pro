# Sample Images for Home Screen

Place your sample images here. The HomeScreen expects these files:

## Category Cover Images (shown on main cards)
- `image-cover.jpg` — Image category hero (e.g. a beautiful AI-generated portrait)
- `video-cover.jpg` — Video category hero (e.g. a cinematic video still)
- `audio-cover.jpg` — Audio category hero (e.g. waveform visualization or headphones)
- `transcribe-cover.jpg` — Transcribe category hero (e.g. microphone or speech bubbles)
- `character-cover.jpg` — Custom Character hero (e.g. AI-generated character)

## Sub-tool Thumbnails (shown in NEXUS AI Suite grid)
- `create-image.jpg` — Create Image tool thumbnail
- `edit-image.jpg` — Edit Image tool thumbnail
- `create-video.jpg` — Create Video tool thumbnail
- `animate-image.jpg` — Animate Image tool thumbnail
- `motion-sync.jpg` — Motion Sync tool thumbnail
- `tts.jpg` — Text to Speech thumbnail
- `music-gen.jpg` — Music Generation thumbnail
- `transcribe.jpg` — Transcribe Audio thumbnail
- `train.jpg` — Train Model thumbnail

## Recommended Size
- Category covers: 400x500px (portrait ratio, tall cards)
- Sub-tool thumbnails: 200x200px (square)
- Format: JPG or PNG, keep under 200KB each

If images are missing, a gradient placeholder with an emoji icon will be shown instead.
